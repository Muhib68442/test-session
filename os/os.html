OS Installation (Windows)

In this page, we will learn how to install a windows properly in your PC/Laptop.

Before Installation, make sure you BACKED UP all your data otherwise any data may be lost

Make sure you properly PLUGGED IN your Laptop Charger. Powering down the system between windowsinstallation or update may damage your device

Windows Requirements (Windows 10) (Minimum)

  • Processor : 2GHz+
  • RAM : 4GB+
  • Storage Scace : 25GB+

Step-1 : Download Windows Media Creation Tool (Windows 10), Then Open it.

Step-2 : Press Accept and accept microsoft notices and terms

Step-3 : Choose Installation Media and Click Next

Step-4 : Select Edition or let the recommended setting and click Next

Now, you have to choose media. if you want to install windows in your active device, Choose ISO File. Oterwise Select USB Flash and make a pendrive bootable.

Install from Windows

If your Device have an Active Windows :

Step-1 : Click Download and download the windows package file

Step-2 : Open the folder where the file downloaded

Step-3 : Open the Windows.iso file, inside that, open the setup.exe

Install from a Bootable Device

To install windows from a bootable media, you need a Pendrive at least 8GB.

(NOTE : Your pendrive will be formatted. all data will be erased in this process. you can recover your pendrive later.)

Step-1 : Select USB Flash Option

Step- : Let the windows file download and make a bootable drive

Step- : After completing, remove the USB Drive and plug-in to the disabled computer.

Step- : Turn on the Computer, after BIOS page, hit any key to enter the setup...

Step- : Setup Started !
