OS Installation (Windows)

In this page, we will learn how to install a windows properly in your PC/Laptop.

NOTE : Before Installation, make sure you BACKED UP all your data otherwise any data may be lost

Make sure you properly PLUGGED IN your Laptop Charger. Powering down the system between windows installation or update may damage your device

Windows Requirements (Windows 10) (Minimum)

  • Processor : 2GHz+
  • RAM : 4GB+
  • Storage Scace : 25GB+

Step-1 : Download Windows Media Creation Tool (Windows 10), Then Open it.

Step-2 : Press Accept and accept microsoft notices and terms

Step-3 : Choose Installation Media and Click Next

Step-4 : Select Edition or let the recommended setting and click Next

Now, you have to choose media. if you want to install windows in your active device, Choose ISO File. Oterwise Select USB Flash and make a pendrive bootable.

Install from Windows

If your Device have an Active Windows :

Step-1 : Select "ISO File", Click Next and download the Windows Installation File (Windows.iso)

Step-2 : Open the folder where the file downloaded

Step-3 : Open the Windows.iso file, inside that, open the setup.exe

Step-4 : A new Windows 10 Setup window will open, Click Next

Step-5 : Click Accept and Accept the Terms ad Conditions

Step-5 : After Completing check for updates, Click Install

Install from a Bootable Device

To install windows from a bootable media, you need a Pendrive at least 8GB.

(NOTE : Your pendrive will be formatted. all data will be erased in this process. you can recover your pendrive later.)

Step-1 : Select USB Flash Option

Step-2 : Let the windows file download and make a bootable drive

Step-3 : After completing, remove the USB Drive and plug-in to the disabled computer.

Step-4 : Turn on the Computer, after BIOS page, there might be blinking "Press any key to boot from CD or DVD...". Hit any key to enter the setup...

Step-5 : Setup Started !

Setup Process

Step-1 : In windows setup screen, select Language and click Next

Step-2 : Click Install Now

Step-3 : in the "Activate Windows" Screen, we will select "i don't have a product key" and bypass the activation

Step-4 : in the next screen, select the Windows Version and click next

Step-5 : in Licence Terms, click Next

Step-6 : in next screen, select Custom for fresh windows installation

Step-7 : Select Type:Primary drive Partition (C Drive) where was the previous windows installed.

Note : if you want to clear C drive and create a fresh windows, click Format below, otherwise you can continue and select Next

Step-8 : Now windows installation will be started.

Note : Your PC may be restart for several times

After completing steps, remove usb drive if inserted

After some time, your windows will complete installation and now you can setup and personalize your windows.
