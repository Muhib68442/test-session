More tips to Optimize and get back Performance

  • Restart : Helps resolve issues and improve performance by clearing memory and stopping problematic processes.
    • Click on the Windows icon > power icon > "Restart".

  • Enable Virtualization : Allows multiple operating systems on one computer.
    • Restart computer > enter BIOS setup (`F2`, `F10`, or `Del`) > enable "Virtualization Technology" or "Intel VT-x" > save changes and exit.

  • Pause OneDrive Sync : Temporarily stops syncing files to the cloud.
    • Right-click on OneDrive icon in system tray > "Pause syncing" > choose duration.

  • Cleaning up system drive : Frees up space for better performance.
    • Right-click on primary drive (C:) > "Properties" > "Disk Cleanup" > select files to delete.

  • Uninstalling Program : Removes unwanted software.
    • Open Control Panel > "Programs and Features" > right-click on program > "Uninstall".

  • Power Setting : Adjusts power usage for better performance or battery life.
    • Click on Windows icon > gear icon > "System" > "Power & sleep".

  • Disabling Visual Effects : Improves performance by reducing visual load.
    • Right-click on Windows icon > "System" > "Advanced system settings" > "Performance" section > "Settings" > choose effects to disable.

  • Closing Unused running apps : Frees up memory and processing power.
    • Open Task Manager (`Ctrl + Shift + Esc`) > "Processes" tab > right-click on app > "End task".

  • Cleaning Browser Cache : Frees up space and can improve browser performance.
    • Open browser settings/options menu > "Clear browsing data" > check "Cached images and files" box > clear data.

  • Disabling unused browser extension : Reduces browser load for better performance.
    • Open browser settings/options menu > "Extensions" > toggle off unused extensions.

  • Cleaning up Hardwares and Fan : Removes dust and debris for better cooling and performance.
    • Shut down and unplug computer > open case > use compressed air to blow out dust/debris.

Advance Tools (CMD)

  • Disk Cleanup (Advance):
    • cmd.exe /c Cleanmgr /sageset:65535 & Cleanmgr /sagerun:65535
  • Check Battery Status :
    • powercfg /batteryreport
  • Check Ping :
    • ping 192.168.0.1 -t
  • System File Checker :
    • sfc /scannow
  • Reset DNS Cache :
    • ipconfig /flushdns
  • Check Disk Errors :
    • chkdsk
